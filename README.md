# Mock Stock

A representation of the userâ€™s stock portfolio. The user will be able to see stock holdings, portfolio balance, stock-specific information, and be able to search for current stock data. The app will have full CRUD capabilities in that the user can add new stocks upon purchase, see their portfolio information, update their portfolio upon change to holdings (buy/sell in part), and delete stocks upon sale. 

## Technologies used:
- JavaScript
- Express
- Node.js
- Mongoose
- React
- Fetch & Axios
- Bootstrap/React Bootstrap
- CSS Flexbox
